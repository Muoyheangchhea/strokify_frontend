Common Solutions

  • Check if your backend server is running - Your server is running on port 5000 ✅
  • Verify the endpoint URL - Make sure your endpoint is at /api/symptoms/detect
  • Check your symptomRoutes.js - Ensure the route is properly defined
  • Validate your data format - Ensure the request payload matches what your backend expects
  • Check for any middleware issues - Your CORS setup looks correct

Route Configuration Check

In your routes/symptomRoutes.js, make sure you have:

const express = require('express');
const router = express.Router();
const { detectSymptoms } = require('../controllers/symptomController');

router.post('/detect', detectSymptoms);

module.exports = router;
